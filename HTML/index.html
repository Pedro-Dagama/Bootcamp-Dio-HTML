Surgimento da internet

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa HyperText Markup Language, traduzindo ao português: Linguagem de Marcação de Texto. O HTML é componente fundamental da WEB, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações em uma página web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens, vídeos, documentos, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimento e guardando informações.

Ao visitar uma página simples na web você pode perceber que existem diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e qualquer outro elemento. Essa estrutura é estabelecida através do HTML. No início da web era comum encontrar sites apenas contendo textos e imagens simples, com estrutura básica e sem estilizações. Porém, nos dias atuais, muito dificilmente você encontrará sites que possuam apenas elementos HTML. Portanto, podemos considerar o HTML o "esqueleto" da sua página.

Imagine então que além do esqueleto é necessário ter corpo. Para isso, temos então as linguagens CSS e o JavaScript, que em conjunto com o HTML formam a base para todos os websites atuais. Veremos mais à frente o que significam essas linguagens.

Como funciona o HTML

O HTML (HyperText Markup Language) funciona como a linguagem que estrutura o conteúdo da web. Ele é interpretado pelo navegador, que transforma as marcações em textos, imagens, links e outros elementos visíveis em uma página.

Um documento HTML começa com a declaração do tipo de documento (<!doctype html>), seguido pela tag <html>. Dentro dela, temos o <head> (com informações como título e metadados) e o <body>, que contém o conteúdo mostrado ao usuário.

O HTML é composto por tags, que indicam a função de cada parte do conteúdo. Por exemplo: <h1> é usado para títulos principais, <p> para parágrafos e <img> para imagens.

O navegador lê o código HTML, constrói uma estrutura em árvore chamada DOM (Document Object Model) e apresenta o conteúdo na tela. Em conjunto com o CSS (estilo) e o JavaScript (funcionalidades), o HTML forma a base de todos os sites.

O que são Tags

No HTML, tags são palavras-chave usadas para indicar o papel ou a função de um conteúdo dentro da página. Elas são escritas entre sinais de menor e maior, por exemplo: <p> representa um parágrafo, <h1> um título principal e <img> uma imagem.

Existem dois tipos principais de tags:

1. Tags de abertura e fechamento: Elas delimitam um elemento e contêm conteúdo entre elas. Exemplo: <p>Este é um parágrafo</p>.

2. Tags auto-fecháveis: Elas não possuem conteúdo interno e não precisam de fechamento. Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">.

As tags podem ter atributos, que fornecem informações extras ou definem propriedades do elemento. Por exemplo, a tag <a> pode ter o atributo href para indicar o link de destino: <a href="https://www.exemplo.com">Visite o site</a>.
